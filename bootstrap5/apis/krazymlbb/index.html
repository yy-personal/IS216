KrazyMLBB API v1.0 - Documentation

read all

Description
Returns a collection of MLBB heroes in JSON format.

HTTP request
GET http://krazywoman.com/krazymlbb/api/hero/read.php

read one

Description
Given an ID value, returns an MLBB hero in JSON format.

HTTP request
GET http://krazywoman.com/krazymlbb/api/hero/read_one.php?id=19
where id is the id parameter, and
19 is the id value.

read by page

Description
Returns a collection of MLBB heroes in JSON format - by page.
Each page displays 5 heroes at a time.
e.g. Page 1 displays heroes with id between 1 and 5, page 2 displays heroes with id between 6 and 10, and so on.

HTTP request GET http://krazywoman.com/krazystars/api/star/read_paging.php?page=3

search

Description
Returns a collection of MLBB heroes of a certain class in JSON format.

HTTP request
GET http://krazywoman.com/krazymlbb/api/hero/search.php?c=mage
where c is the class parameter, and
mage is the value (e.g. mage, fighter, support, tank, assassin, marksman).
