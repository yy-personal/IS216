KrazyStars API v1.0 - Documentation

read

Description
Returns a collection of stars in JSON format.

HTTP request
GET http://krazywoman.com/krazystars/api/star/read.php

search

Description
Returns a collection of stars matching the search query in JSON format.

HTTP request

  • Search all stars whose gender is female
    GET http://krazywoman.com/krazystars/api/star/search.php?g=f
    where g is the gender parameter, and
    f (or F) indicates female
    m (or M) indicates male

  • Search all stars whose name contains substring
    GET http://krazywoman.com/krazystars/api/star/search.php?n=emm
    where n is the name parameter, and
    emm is the substring
    The above GET request will return Emma Watson and Emma Stone.

  • Search all stars whose name contains substring AND gender is female/male
    GET http://krazywoman.com/krazystars/api/star/search.php?n=ry&g=m
    where n is the name parameter, and ry is the substring
    AND g is the gender parameter, and m (or M) indicates male The above GET request will return Ryan Gosling and Henry Cavill.
